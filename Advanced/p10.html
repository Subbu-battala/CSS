CSS PSEUDO-CLASSES..!

the css pseudo-classes are allow you to style the synamic states of an element such as hover, active and focus state,as well as element that are existing in the document tree but cant be targetted via the use of other selectors without adding any ids or classes to them , for example , targetting the first or last child elements.

A pseudo-class starts with a colon(:).its syntax can be given with:

selector:pseudo-class { property: value;}
