CSS Pseudo-elements..!

the css Pseudo-elements allow you to style the elements or parents of the elements without adding any ids or classes to them

css introduced a new double colon (::) syntax for Pseudo-elements to distinguish between them and Pseudo-classes

selector::Pseudo-element {property value; }
